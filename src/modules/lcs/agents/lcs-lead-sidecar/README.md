# lcs-lead-sidecar

This folder stores persistent memory for the **Conrad (LCS Lead)** agent.

## Purpose

Persist consolidation state across sessions: current phase, audit results, safety nets status, coverage metrics, decision log with justifications, and history.

## Files

- `memories.md` — Project context, session history, decision log
- `instructions.md` — Startup protocol, operational rules, auto-triggers, communication patterns

## Runtime Access

After BMAD installation, this folder will be accessible at:
`{project-root}/_bmad/_memory/lcs-lead-sidecar/`
